# ClickPost (클릭포스트) 초정밀 개발 로드맵 (v2.2)

본 문서는 `gemini.md` 가이드라인에 따라 ClickPost 프로젝트를 완성하기 위한 50단계의 초정밀 개발 프로세스를 정의합니다. 현재까지의 진행 상황을 반영하고 있으며, 향후 "다음 단계를 진행해 주세요"라는 요청에 따라 즉각적인 개발이 가능하도록 설계되었습니다.

## 🟢 Phase 1: 기반 아키텍처 및 코어 AI 모듈 (완료)
1. **[완료]** Supabase DB 스키마 마이그레이션 (ERD v2.1 반영)
2. **[완료]** Users 및 Avatars 테이블 최적화 (Seed ID 시스템 구축)
3. **[완료]** Campaigns 테이블 생성 (예산, 타겟 필터, 키워드 필드)
4. **[완료]** NanoBanana API 연동 기초 (모바일 아바타 에셋 생성)
5. **[완료]** 아바타 스튜디오 고도화 (Supabase Storage 연동)
6. **[완료]** Gemini API 자료 분석 엔진 개발 (광고주 자료 분석)
7. **[완료]** 광고주용 반응형 웹 기초 UI (Bento Grid 스타일 적용)
8. **[완료]** 캠페인 빌더 구현 (정보, 타겟팅, 키워드 입력 단계)
9. **[완료]** AI 스크립트 샌드박스 (Gemini 1.5 Flash 연동 및 3종 생성)
10. **[완료]** LBS 타겟팅 인프라 (PostGIS 기반 위치 쿼리 준비)

12. **[진행 예정]** 에스크로 결제 시스템 연동 (PayPal/Stripe 연동)

14. 승인 스크립트 기반 100종 변주 생성 배치 (Gemini Caching 기술 적용)

16. Google Veo v2 Ultra 전용 프롬프트 빌더 고도화 (스크립트+아바타 결합)

18. 모바일 미션 마켓플레이스 UI (플랫폼별/단가별 필터링 기능)

20. 미션 선점(Preemption) 및 만료(TTL) 시스템 구현 (중복 참여 방지)

23. 생성된 720p 영상의 Supabase Storage 자동 업로드 및 경로 저장
24. 영상 스트리밍/다운로드 최적화 (CDN 캐싱 전략 적용)
25. SNS 업로드 가이드 및 해시태그 복사 기능 UI 구현
