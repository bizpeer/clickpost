# ClickPost (클릭포스트) 상세 개발 로드맵 (v2.1)

본 문서는 `gemini.md` 가이드라인에 따라 ClickPost 프로젝트를 완성하기 위한 30단계의 구체적인 개발 프로세스를 정의합니다.

## Phase 1: 기반 아키텍처 및 코어 AI 모듈 (Steps 1-10)
1. **Supabase/Firebase DB 스키마 마이그레이션**: ERD v2.1 기준 테이블 생성 (PostGIS 확장 활성화 포함).
2. **Users 및 Avatars 테이블 최적화**: 인덱싱 및 고정 시드(Seed ID) 필드 구성.
3. **Campaigns 테이블 생성**: 예산, 수수료율, 타겟 필터, 필수/제외 키워드 필드 추가.
4. **NanoBanana API 연동 (Mobile)**: 가입 정보 기반 5개 각도(정면, 측면 등) 아바타 에셋 생성 파이프라인 구축.
5. **아바타 스튜디오 고도화**: 생성된 에셋의 클라우드 스토리지(Supabase Storage) 저장 및 조회 로직 구현.
6. **Gemini API 기반 자료 분석 엔진**: 광고주 업로드 자료(이미지/PDF) 및 키워드 분석 로직 개발.
7. **광고주용 반응형 웹 기초 UI**: Next.js 기반 캠페인 관리 대시보드 및 빌더 레이아웃 설계.
8. **캠페인 빌더 구현**: 제품 이미지 업로드, 타겟팅(지역, 성별, 연령) 설정 UI 개발.
9. **AI 스크립트 샌드박스 기초**: Gemini 연동을 통한 3종 샘플 스크립트 실시간 생성 UI.
10. **LBS 타겟팅 인프라**: PostGIS `geometry` 필드를 활용한 회원 위치 기반 필터링 쿼리 작성.

## Phase 2: 캠페인 파이프라인 및 온디맨드 영상 (Steps 11-20)
11. **스크립트 검토 및 승인 프로세스**: 광고주의 샘플 수정/확정 및 캠페인 상태 전환 로직.
12. **에스크로 결제 및 예산 예치**: 결제 완료 시 캠페인 'ACTIVE' 전환 및 예산 홀딩 시스템.
13. **대량 스크립트 파생 엔진 (Caching)**: 승인된 샘플 기반 100종 변주 스크립트 사전 생성 및 DB 저장.
14. **Google Veo 프롬프트 자동 생성**: 각 스크립트와 회원 아바타 에셋을 결합한 Veo 전용 프롬프트 빌더 개발.
15. **미션 리스트 및 필터링 (Mobile)**: LBS 및 유저 속성 매칭을 통한 개인화된 미션 노출 로직.
16. **미션 선점(Preemption) 시스템**: 참여 클릭 시 스크립트 할당 및 만료 시간(`expires_at`) 설정 (중복 방지).
17. **JIT(Just-In-Time) 영상 생성 연동**: 모바일 요청 시점에 Google Veo API 호출 및 720p 렌더링.
18. **영상 처리 및 스토리지 최적화**: 생성된 영상의 자동 업로드 및 모바일 스트리밍 최적화.
19. **SNS 가이드 제공 UI**: 할당된 스크립트, 해시태그 복사 및 업로드 가이드 화면 구현.
20. **미션 참여 이력 관리**: 유저별 진행 중/완료/만료 미션 상태 트래킹.

## Phase 3: 사후 검증 및 정산 자동화 (Steps 21-30)
21. **미션 제출(Submission) API**: SNS 게시물 URL 제출 및 기본 유효성 검사 로직.
22. **URL 스크래핑 및 1차 검증**: 필수 키워드 포함 여부 및 게시물 존재 여부 즉시 확인.
23. **1차 포인트 지급 트랜잭션**: 검증 완료 직후 캠페인 예산에서 유저 포인트로 즉시 전환.
24. **45일 유지 추적 배치 시스템**: 매일 전체 제출 URL을 순회하며 삭제 여부를 체크하는 자동화 스케줄러.
25. **보너스 지급 로직**: 45일 유지 성공 시 `bonus_45d` 포인트 자동 지급 및 정산 완료 처리.
26. **글로벌 환율 및 페이먼트 연동**: 실시간 환율 적용 및 현지 통화(Paypal, NaverPay 등) 정산 엔진.
27. **출금 신청(Payout) 관리**: 유저의 출금 신청, 심사, 승인 및 외부 결제망 연동 처리.
28. **관리자 통합 대시보드**: 전체 광고 집행액, 수익(수수료), API 비용 실시간 모니터링 트래커.
29. **비용 방어 모드 (Cost Defense)**: API 하드 리밋 및 예산 소진 시 미션 자동 마감 기능 활성화.
30. **최종 통합 테스트 및 배포**: 전체 시나리오(광고주 등록 ~ 유저 정산) 테스트 및 프로덕션 환경 배포.
