# ClickPost (클릭포스트) 초정밀 개발 로드맵 (v2.3)

본 문서는 `gemini.md` 가이드라인에 따라 ClickPost 프로젝트를 완성하기 위한 초정밀 개발 프로세스를 정의합니다. 현재까지의 진행 상황(GitHub Pages 멀티 앱 배포 등)을 반영하고 있으며, 향후 개발 재개 시 즉각적으로 이행해야 할 잔여 과제들을 상세히 기록합니다.

---

## 🟢 Phase 1 & 2.1: 기반 아키텍처 및 광고주 코어 (95% 완료)
1. **[완료]** Supabase DB 스키마 마이그레이션 (ERD v2.1 반영)
2. **[완료]** Users 및 Avatars 테이블 최적화 (Seed ID 시스템 구축)
3. **[완료]** Campaigns 테이블 생성 (예산, 타겟 필터, 키워드 필드)
4. **[완료]** 광고주용 반응형 웹 기초 UI (Bento Grid 스타일 적용)
5. **[완료]** 캠페인 빌더 구현 (정보, 타겟팅, 키워드 입력 단계)
6. **[완료]** AI 스크립트 샌드박스 (Gemini 3 Flash 연동 및 3종 생성/편집/승인)
7. **[완료]** 필수 키워드 포함 여부 자동 검증 로직 구현
8. **[완료]** GitHub Pages 멀티 앱 배포 환경 구축 (광고주 웹: `/`, 관리자 웹: `/admin`)
9. **[진행 예정]** **실시간 키워드 하이라이팅 UI**: 샌드박스 내 편집 시 필수 키워드 강조 기능.

## 🟡 Phase 2.2 & 2.3: 백엔드 확장 및 온디맨드 영상 (85% 완료)
10. **[완료]** Supabase Edge Function: 승인 스크립트 기반 100종 변주 생성 배치
11. **[완료]** Google Veo 전용 영상 제작 프롬프트 자동 생성기 (Script Variations 통합)
12. **[완료]** LBS 타겟 필터링 엔진 (PostGIS 쿼리 최적화)
13. **[진행 예정]** **프리미엄 미션 초대 시스템**: 10K+ 인플루언서 대상 푸시 알림 및 초대 로직.
14. **[완료]** 인플루언서 미션 마켓플레이스 UI (모바일 앱 연동)
15. **[완료]** JIT(Just-In-Time) 영상 생성 연동 (참여 클릭 시 Veo API 트리거)
16. **[진행 중]** **미션 만료 타이머 및 선점 취소 시스템**: Ghost Mission 방지 로직.
17. **[진행 중]** **실시간 미션 매칭 알림(FCM) 연동**: 타겟팅 지역 진입 시 알림 발송.
18. **[진행 예정]** **NanoBanana 아바타 에셋 결합**: Seed ID 기반 고정 아바타 합성 로직 고도화.

## 🚀 개발 재개 시 실행 가이드

사용자가 **"다음 단계를 진행해 주세요"**라고 요청할 경우, AI는 아래 순서에 따라 즉시 개발을 재개합니다.

1. `ROADMAP.md`에서 현재 `[진행 중]` 또는 `[진행 예정]`인 최우선 과제(Step 9, 13, 16 등)를 확인합니다.
2. `gemini.md`의 핵심 비즈니스 로직(아바타 고정, 수수료 구조 등)을 재검토합니다.
3. 현재의 배포 환경(`GitHub Pages`, `Supabase Edge Functions`)을 바탕으로 추가 기능을 구현합니다.
4. 구현 완료 후 `task.md`와 본 문서를 업데이트하여 진척도를 관리합니다.
